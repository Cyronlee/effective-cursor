# Introduction

> Choose the most AI-friendly tech stack

If your project already has a determined tech stack, feel free to skip this chapter. The next chapter will cover how to add AI assistance to existing projects.

Choosing the right tech stack not only affects development efficiency but also directly impacts the effectiveness of AI-assisted programming. This chapter analyzes which technologies are more AI-friendly and helps you make informed technology decisions.
