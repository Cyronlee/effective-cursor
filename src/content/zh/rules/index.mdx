# Cursor Rules 模板库

<Callout type="info">
这里收录了经过验证的 Cursor Rules 模板，可以直接复制使用或作为参考进行定制。
</Callout>

## 快速开始

1. **选择模板** — 在下方选择适合你技术栈的规则集
2. **复制规则** — 点击规则页面中的复制按钮，将内容复制到 `.cursor/rules/` 目录
3. **按需调整** — 根据项目实际情况修改规则内容

> 💡 如果你还不了解 Cursor Rules 的工作原理和编写方法，建议先阅读 [Cursor Rules 指南](/zh/docs/3-cursor-rules)。
