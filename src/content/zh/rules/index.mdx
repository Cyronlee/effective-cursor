# Cursor Rules 规则集合

<Callout type="info">
这里收录了不同项目类型的 Cursor Rules 配置，可以作为你项目的参考模板。
</Callout>

## 什么是 Cursor Rules？

Cursor Rules 是 Cursor AI 编辑器中的项目级配置规则，通过在 `.cursorrules` 文件中定义规范，可以让 AI 更好地理解你的项目结构、技术栈和编码风格，从而生成更符合项目要求的代码。

## 如何使用？

1. **浏览规则集**：在左侧导航栏选择你的项目类型（如 Next.js + shadcn/ui）
2. **查看规则详情**：每个项目类型包含多个规则文件，涵盖不同方面的规范
3. **复制到项目**：使用 Playground 组件中的复制按钮，将规则内容复制到你的 `.cursorrules` 文件中
4. **按需调整**：根据实际项目需求，修改和定制这些规则

## 规则分类

### 全局规则 (Global Rules)
- 适用于整个项目的通用规范
- 包括技术栈、目录结构、编码标准等
- 通常设置 `alwaysApply: true`

### 特定规则 (Specific Rules)
- 针对特定文件类型或目录的规则
- 通过 `globs` 字段指定应用范围
- 如路由规则、UI 组件规则等
