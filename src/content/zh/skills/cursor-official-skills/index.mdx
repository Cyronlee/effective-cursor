## 如何使用官方 Skills

### 方式一：自动触发

Agent 会根据上下文自动判断是否需要调用某个 Skill。例如，当你说"帮我创建一个新的 rule"时，Agent 会自动使用 `create-rule` Skill。

### 方式二：手动调用

在 Agent 对话框中输入 `/`，然后搜索 Skill 名称即可手动调用：

```
/create-rule
```

### 方式三：禁用自动调用的 Skills

某些 Skills 设置了 `disable-model-invocation: true`，这意味着它们不会被自动调用，只能通过 `/skill-name` 的方式显式触发。例如：

- `create-subagent`
- `migrate-to-skills`

<Callout type="warning">
**注意**: 请勿修改 `~/.cursor/skills-cursor/` 目录中的内容，这些文件由 Cursor 自动管理。如果你想创建自定义 Skills，请使用 `.cursor/skills/`（项目级）或 `~/.cursor/skills/`（用户级）目录。
</Callout>
