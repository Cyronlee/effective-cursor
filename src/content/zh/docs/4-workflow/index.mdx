# 引言

> 把 AI 深度融入日常开发流程，提升开发效率与代码质量

## 核心关注点

- **Tasking, designing, coding for accuracy** — 任务拆分、方案设计、精准编码
- **AI-assisted development, human-focused innovation** — AI 辅助开发，人类主导创新

在前面的章节中，我们通过 Cursor Rules 约束了 AI 生成代码的边界，并配置了自动化检测和修复。有了这些保障，下面介绍的工作模式都能产出高质量的代码。

## 本章目标

通过本章学习，你将掌握：

- 理解四种 Cursor 使用模式的**适用场景**
- 学会根据任务复杂度**选择合适的模式**
- 掌握 **Draft-Final** 模式进行复杂任务开发
- 建立从需求到上线的**标准工作流程**
