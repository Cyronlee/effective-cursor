# AI 友好的技术栈

> 从"写代码"到"描述意图"——选择能与 AI 对话的技术

在 Cursor、Copilot、Claude 等 AI 辅助开发工具时代，技术选型有了新的维度：**AI 友好性**。

LLM 不像人类那样写代码，而是通过 **自然语言 → 语义推断 → 结构化输出** 的方式工作。这意味着：

- **歧义越少 = 生成质量越高**
- **结构越显性 = 重构成本越低**
- **表达越声明式 = AI 理解越准确**

正确的技术栈不仅提升你的生产力，更从根本上决定了 AI 能多有效地帮助你。本章分析哪些技术真正对 AI 友好，帮助你做出明智的决策。
